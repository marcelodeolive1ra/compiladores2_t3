Criação de uma linguagem de programação e construção de um compilador para esta linguagem.

No caso, a linguagem criada é uma linguagem para a criação de websites incríveis, como este :-)

Projeto completo deste compilador disponível no GitHub:

http://github.com/marcelodeolive1ra/compiladores2_t3

O que você pode fazer com esta linguagem?

Banner

Crie um banner como o desta página com banner { titulo(...) subtitulo(...) }

Menus

Criar menus é muito fácil: use menu { item(...) -> link }. Caso queira que algum link abra em uma nova aba, basta usar ->+.

Sidebars

Use sidebar { item() } para criar uma sidebar e automaticamente adicionar um botão para ativá-la no menu principal.

Imagens

Basta usar imagem(local). Caso queira um link, é só usar a mesma sintaxe de links em itens de menus.

Você pode organizar o conteúdo em colunas

E em qualquer string, pode usar as tags clássicas de HTML, como <strong> para texto em negrito, <i> para texto itálico e <u> para texto sublinhado. Pode também usar a tag <br> para forçar uma quebra de linha dentro de um parágrafo.

Pode usar subtítulos

E assim criar um texto longo com a combinação de títulos/subtítulos e parágrafos.

Pode criar uma seção com várias colunas

E alinhá-las como quiser

E quando a seção é composta por colunas, pode adicionar texto no início da seção.
E é possível utilizar alinhamento à esquerda, centralizado e à direita para o conteúdo das colunas.
